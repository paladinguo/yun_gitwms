# 吉特仓储管理系统-云服务版本
该版本将Web显示端与API端做了分离,新增了多仓库，多企业，采购销售，以及财务部分模块。 便于二次开发，可以快速应对一部分中小型客户需求，2018年个人所开发的所有项目均在此基础上深化而来。
